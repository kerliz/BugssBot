# Bugss - Discord bot

Just a simple bot to annoy you

## How to run
* Clone the repo
* `npm i discord.js` to install dependencies
* In bot.js file replace `TOKEN` with your own Discord token
* To run on your local server: `node bot.js`
* And bot is ready to annoy everyone

### Trigger Words
* I
* Team
* Welcome
* If
* Discord
* We
* Can
* But
* And
* Code
* Dev
* Cat
